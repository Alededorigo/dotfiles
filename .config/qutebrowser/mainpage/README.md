## Set up

### Extensions:
* For Firefox use "Custom New Tab Page"
* For Chrome use "Custom New Tab"

### Usage:
On the new tab: use one of the extensions mentioned

On the home page:
* Firefox: Go into
```Preferences > Home > Homepage and new windows```
* Chrome: Go into
``` Options > Start > Homepage```
